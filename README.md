# TO-DOIT - Todo List Application

A modern, feature-rich todo list application built with HTML, CSS, and JavaScript. Styled with Tailwind CSS and DaisyUI for a beautiful, responsive interface.

## Features

- **Task Management**: Add, edit, delete, and mark tasks as complete
- **Subtasks**: Create subtasks under parent tasks for better organization
- **Due Dates**: Set due dates for tasks to stay on schedule
- **Progress Tracking**: Visual progress bar and statistics showing completion status
- **Search & Filter**: Search tasks and filter by status (All, Pending, Completed)
- **Drag & Drop**: Reorder tasks with drag and drop functionality
- **Theme Switcher**: Choose from 11 beautiful themes (Cupcake, Dark, Light, Bumblebee, Synthwave, Halloween, Fantasy, Dracula, Aqua, Luxury, Night)
- **Keyboard Shortcuts**: Quick access with keyboard shortcuts
- **Local Storage**: Tasks persist in browser local storage
- **Responsive Design**: Works on desktop and mobile devices

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | Add new task |
| `Ctrl + F` | Search tasks |
| `Ctrl + A` | Show all tasks |
| `Escape` | Cancel operation |
| `Enter` | Add task (from input) |

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start adding tasks!

## Files

- `index.html` - Main HTML structure
- `style.css` - Custom styles
- `main.js` - Application logic (TodoManager, UIManager, ThemeSwitcher)
